# Vue 3 + Vite



使用说明
正常情况下有三种业务流程：
流程一、新租户入住
1.租户管理模块：录入租户信息，包括姓名，电话，身份证等
2.房间管理模块：在对应的房间点击“入住”按钮，并选择刚录入的租户，保存，此时会在租房记录模块自动生成一条记录
3.水电表管理模块：点击右上角“新增记录”按钮，将水电表读数录入到系统，记录类型选择新租户入住

流程二、日常收租
1.本月应收租模块：查看本月应该收取租金的房间，未出租状态的房间不显示，已收租的房间状态显示为已缴费
2.水电表管理模块：点击右上角“新增记录”按钮，选择需要收租的房间，将当前水电表读数录入到系统，此时系统会根据水电表上一次的读数计算出水电用量，并自动在缴费管理模块自动生成一条未缴费的记录，包含房租，水费，电费等各项费用
3.缴费管理模块：核对各款项无误后，自行向租户收取缴款，在确认收到租户的缴款之后，点击“缴费按钮”，核对各项金额无误后，点击保存

流程三、退房
1.水电表管理模块：将用户退房时的水电表读数录入系统，此时系统会自动判断这个租户本月是否已经交过房租来生成缴款记录
2.缴费管理模块：核对各款项无误后，自行向租户收取款项，在确认收到租户的缴款之后，点击“缴费按钮”，核对各项金额无误后，点击保存
3.房间管理模块：在对应的房间点击“退房”按钮，保存，此时系统会自动更新对应的租房记录上的退房时间，并把改房间标记为空置状态


另外，水电用量统计模块可以查看本年度各个房间的水电用量排行，以及每个月的水电用量统计
